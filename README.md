This is a full stack contact form application built using Java Spring Boot and HTML/CSS/JavaScript.

## Features
- Modern and responsive contact form UI
- Live email validation with custom messages
- Animated submit button with loading indicator
- Inline success message (no page redirect)
- Backend built with Spring Boot
- Form data submitted using Fetch API

## Technologies Used
- Java
- Spring Boot
- HTML
- CSS
- JavaScript

## How to Run

### Backend
1. Open the backend project in IntelliJ
2. Run the Spring Boot application
3. Server runs on `http://localhost:8080`

### Frontend
1. Open `index.html` using Live Server in VS Code
2. Fill the form and submit
