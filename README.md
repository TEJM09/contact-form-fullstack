**Java Full Stack Web Development Internship at Maincrafts Technology**.

---

## Features

- Modern and responsive contact form UI  
- Live email validation with custom error messages  
- Animated submit button with loading indicator  
- Inline success message without page redirection  
- Backend developed using Spring Boot  
- REST API for handling form submissions  
- Data persistence using JPA & Hibernate  
- Automatic table creation in MySQL  
- Successful storage and verification of submitted data

---

## How to Run the Project

### Backend
1. Open the backend project in IntelliJ IDEA  
2. Configure MySQL database details in `application.properties`  
3. Run the Spring Boot application  
4. Server runs on `http://localhost:8080`

### Frontend
1. Open `index.html` using Live Server in VS Code  
2. Fill in the contact form and submit  
3. Submitted data will be stored in the MySQL database  

---

## Database Verification

- MySQL table is automatically created using Hibernate  
- Submitted form data can be verified using SQL queries
